# QVToolkit Prototype
- [ ] Login via Github, Twitter or Metamask.
- [ ] Grab the list via config files from Github.
- [ ] Everyone who have a ticket will have 10 vote, guest will have 1 vote.

## How to setup
### Client
npm run serve
### Server
node handler.js
### Proxy
node proxy.js
